# Relatório Back-End Pokedex - Overleaf

Este pacote contém o relatório `Relatorio-Back-End-Pokedex` convertido para LaTeX.

## Como usar no Overleaf

1. Acesse o projeto/modelo indicado pelo professor.
2. Faça upload do arquivo `relatorio-back-end-pokedex-overleaf.zip`, ou envie o `main.tex` para dentro do projeto.
3. Defina `main.tex` como arquivo principal, se necessário.
4. Clique em `Recompile`.

## Observação

O link informado do Overleaf não permitiu baixar diretamente os arquivos do template por aqui. Por isso, o relatório foi preparado em LaTeX limpo e compatível com Overleaf. Caso o modelo do link possua arquivos próprios de classe, capa ou estilo, basta manter o conteúdo do `main.tex` e adaptar o cabeçalho conforme o template.
