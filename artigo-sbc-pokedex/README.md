# Artigo SBC - Pokedex em Grafos

Este pacote foi preparado para ser enviado ao Overleaf usando o modelo da Sociedade Brasileira de Computação (SBC).

## Como usar no Overleaf

1. Acesse https://www.overleaf.com/
2. Clique em `New Project`.
3. Escolha `Upload Project`.
4. Envie o arquivo `artigo-sbc-pokedex.zip`.
5. Abra o arquivo `main.tex`.
6. Clique em `Recompile`.

## Arquivos principais

- `main.tex`: artigo científico em LaTeX.
- `sbc-template.sty`: estilo do template SBC.
- `sbc.bst`: estilo bibliográfico SBC.
- `referencias.bib`: referências bibliográficas.
- `figuras/`: imagens usadas no artigo.

## Observação

O artigo está em português e, por isso, contém `abstract` e `resumo`, conforme solicitado no enunciado.
