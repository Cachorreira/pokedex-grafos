# Relatório Docker Completo - Overleaf

Este pacote contém o relatório Docker completo convertido para LaTeX.

## Como usar no Overleaf

1. Acesse o Overleaf.
2. Crie um novo projeto com `Upload Project`.
3. Envie o arquivo `relatorio-docker-completo-overleaf.zip`.
4. Abra `main.tex`.
5. Clique em `Recompile`.

## Conteúdo

- `main.tex`: relatório em LaTeX.
- `figuras/logo-ueg.png`: logo da UEG usada na capa.
- `figuras/*.png`: prints de evidência do Docker, aplicação e Neo4j.

Os nomes dos integrantes foram centralizados na capa, conforme solicitado.
